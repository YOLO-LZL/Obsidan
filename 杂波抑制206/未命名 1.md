
  优先级最高

  - 覆盖性分析
    - 统计每个 az 对应哪些 cpi，画 az-cpi 可用性矩阵。
    - 目的：先确认这是怎样的非规则采样，避免后面把缺失当异常。
  - 维度语义确认
    - 确认 256 x 896 中哪一维是多普勒、哪一维是距离。
    - 从分辨率看，大概率 256 更像多普勒维、896 更像距离维，但最好先用图和物理量核实。
  - 单帧基础统计
    - 对每个 az,cpi 提取总能量、均值、方差、峰值、峰均比、分位数、熵、峰度。
    - 目的：先得到“强杂波帧 / 异常帧 / 稀疏帧”的粗分类。

  最能反映这批数据特征的分析

  - 距离向杂波结构
    - 对每帧沿多普勒维积分，得到距离功率剖面。
    - 看近距强杂波区、远距衰减、局部强散射带、稳定热点距离单元。
    - 输出：平均距离谱、方位分组距离谱、稳定热点距离 bin 列表。
  - 多普勒向杂波结构
    - 对每帧沿距离维积分，得到多普勒功率谱。
    - 重点看：
        - 零多普勒附近能量是否绝对占优；
      - 杂波谱宽有多宽；
      - 是否存在偏移的多普勒中心；
      - 是否出现双峰、拖尾、非对称结构。
    - 输出：多普勒中心、谱宽、零多普勒能量占比、左右不对称度。
  - 二维能量形态
    - 直接看 RD 图的平均图、最大值图、中位数图。
    - 目的：判断杂波是“窄脊线”“宽平台”“局部斑块”还是“多条带状结构”。

  跨方位分析

  - 方位一致性 / 方位差异
    - 比较不同 az 的平均 RD 图、距离谱、多普勒谱。
    - 看哪些方位是强地杂波主瓣区，哪些方位更干净，哪些方位有异常散射源。
    - 这个分析很关键，因为后续虚警率往往强依赖方位。
  - 中心方位 vs 边缘方位
    - 以 az=20 为中心，对称比较 az=20±k。
    - 看能量、谱宽、热点距离是否随方位平滑变化。
    - 如果不平滑，往往意味着地物分布变化或采集几何变化。

  跨 CPI 分析

  - 时间稳定性
    - 在同一 az 下比较不同 cpi 的统计量变化。
    - 看哪些特征稳定、哪些特征波动大。
    - 可提取：
        - 总能量时间序列；
      - 零多普勒占比时间序列；
      - 谱宽时间序列；
      - 热点距离位置是否漂移。
  - 帧间相似度
    - 对同一 az 的不同 CPI 帧做相关系数/余弦相似度。
    - 目的：区分“稳定地杂波背景”与“偶发变化帧”。
  - 异常 CPI 识别
    - 找出明显偏离同方位统计分布的 CPI。
    - 这些帧通常最值得人工看，可能含车辆、处理异常或场景突变。

  面向目标 / 虚警的分析

  - 局部峰检测与峰的可持续性
    - 在每帧做局部峰搜索，但不要急着设严格检测门限。
    - 先统计峰的数量、位置、峰强、是否落在零多普勒附近。
    - 再看这些峰在相邻 CPI 或相邻 az 是否持续。
  - “疑似运动目标”与“稳定杂波峰”分离
    - 稳定杂波峰：常固定在相近距离、相近多普勒、跨 CPI 重复出现。
    - 疑似目标：在 CPI 上有轨迹变化，或在零多普勒外孤立出现。
  - 虚警易发区定位
    - 统计哪些距离-多普勒区域经常出现孤立尖峰。
    - 这会直接告诉你后续 CFAR 或杂波抑制该重点处理哪里。

  建议重点提取的特征

  - 全帧特征
    - 总能量、均值、方差、峰均比、熵、峰度
  - 距离向特征
    - 近距能量占比、热点距离位置、距离谱斜率、有效占据宽度
  - 多普勒向特征
    - 多普勒中心、谱宽、零多普勒占比、左右能量不对称度
  - 二维形态特征
    - 高能区域面积、连通域数量、主脊线宽度、纹理粗糙度
  - 稳定性特征
    - 同 az 下跨 CPI 的均值/标准差/相似度
  - 异常性特征
    - 相对同 az 基线的 z-score、孤立峰数量、异常 bin 占比

  脚本顺序

  - scripts/01_scan_dataset.m
    - 解析 data/ 下所有文件名，提取 az、cpi、路径。
    - 生成一个索引表 dataset_index.mat / dataset_index.csv。
    - 画 az-cpi 可用性矩阵。
    - 目标：先把“不规则采样”看清楚。
  - scripts/02_verify_schema.m
    - 抽样或全量检查每个 .mat 的变量名、尺寸、类型。
    - 确认是否全部都是 CPI_num 和 fft_pc_data_sum，尺寸是否恒定 256x896。
    - 目标：排除脏数据和结构不一致文件。
  - scripts/03_frame_basic_stats.m
    - 对每个文件提取基础统计：
        - 总能量
      - 均值 / 方差
      - 最大值
      - 峰均比
      - 分位数（如 p95/p99）
      - 熵 / 峰度 / 偏度
    - 输出 frame_features.csv。
    - 目标：先把“强杂波帧、异常帧、稀疏帧”分出来。
  - scripts/04_profile_analysis.m
    - 对每帧分别沿两个维度积分，得到两个一维功率谱。
    - 在确认维度语义前，先命名为 dim1_profile、dim2_profile，不要急着写成距离谱/多普勒谱。
    - 画：
        - 全库平均 profile
      - 每个 az 的平均 profile
      - 若干典型帧的 profile
    - 目标：确认哪一维更像多普勒，哪一维更像距离。
  - scripts/05_rd_mean_by_az.m
    - 对每个 az，把其所有可用 cpi 的二维矩阵做平均 / 中位数 / 最大值图。
    - 输出每个 az 的代表性 RD 图。
    - 重点比较：
        - az=20 附近
      - 边缘方位
      - 左右对称方位 20-k 和 20+k
    - 目标：看方位变化如何影响杂波结构。
  - scripts/06_cpi_stability.m
    - 在固定 az 下，分析不同 cpi 的变化：
        - 总能量序列
      - profile 变化
      - 帧间相关系数 / 余弦相似度
      - 与该 az 平均图的偏差
    - 输出每个 az 的稳定性报告。
    - 目标：区分“稳定背景”与“偶发异常”。
  - scripts/07_hotspot_peak_analysis.m
    - 在每帧上做局部峰搜索。
    - 统计：
        - 峰数量
      - 峰强
      - 峰位置
      - 峰是否总出现在固定区域
    - 再按 az 聚合，看哪些位置是稳定热点。
    - 目标：区分稳定杂波热点和疑似运动目标/异常峰。
  - scripts/08_anomaly_ranking.m
    - 用前面提取的特征对每帧打异常分数。
    - 简单做法就够：
        - 同一 az 内做 z-score
      - 或对特征做 PCA 后按重构误差排序
    - 输出每个 az 最异常的前 N 帧。
    - 目标：优先人工看最有信息量的样本。
  - scripts/09_export_feature_bank.m
    - 把可复用的特征统一导出：
        - 全帧特征
      - 两个维度的 profile 特征
      - 稳定性特征
      - 峰类特征
    - 保存成 feature_bank.mat 和 feature_bank.csv。
    - 目标：为后续聚类、检测、杂波抑制建模做准备。
  - scripts/10_review_panels.m
    - 自动生成人工复核图板：
        - 每个 az 的平均图
      - 每个 az 最异常的 3～5 帧
      - 若干稳定热点示例
    - 目标：快速形成“这批数据长什么样”的直观认识。

  最值得先提的特征

  - 全帧级
    - 总能量 sum(X(:).^2)
    - 均值、标准差、最大值
    - 峰均比 max(X(:)) / mean(X(:))
    - 分位数比值，如 p99 / p50
  - 一维谱级
    - 主峰位置
    - 主瓣宽度
    - 能量集中度
    - 左右不对称度
  - 二维形态级
    - 高能区域面积
    - 连通域数量
    - 最大连通域面积
    - 纹理粗糙度
  - 稳定性

  - 异常性
    - 同 az 内的 z-score

    - 局部孤立强点占比
